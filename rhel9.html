RHEL 9 Hardening Controls

SSH Root Login Disabled

Check
grep PermitRootLogin /etc/ssh/sshd_config
Fix
sed -i 's/^PermitRootLogin.*/PermitRootLogin no/' /etc/ssh/sshd_config
Prevents direct root login over SSH for security.

SSH Protocol 2 Enforcement

Check
grep Protocol /etc/ssh/sshd_config
Fix
sed -i 's/^Protocol.*/Protocol 2/' /etc/ssh/sshd_config
Ensures only SSH Protocol 2 is used.

UMASK Setting for root

Check
grep umask /etc/bashrc
Fix
echo "umask 027" >> /etc/bashrc
Sets restrictive default file permissions for root.

Cramfs Kernel Module Blacklist

Check
grep cramfs /etc/modprobe.d/*
Fix
echo "install cramfs /bin/true" > /etc/modprobe.d/cramfs.conf
Disables unnecessary kernel module to reduce attack surface.

User Home Files with Mode > 0750

Check
find /home -xdev -type f -perm /027
Fix
chmod 750 /home/*
Restrict files in user home directories.

Deny Setting in /etc/security/faillock.conf

Check
grep deny /etc/security/faillock.conf
Fix
echo "deny = 5" >> /etc/security/faillock.conf
Lock account after 5 failed login attempts.

GRUB2 Password Config

Check
grep GRUB2_PASSWORD /boot/grub2/user.cfg
Fix
grub2-setpassword
Protect GRUB bootloader with a password.

IP Forwarding Disabled

Check
cat /proc/sys/net/ipv4/ip_forward
Fix
sysctl -w net.ipv4.ip_forward=0
Disables IP forwarding to harden network.

Max Repeat in pwquality.conf

Check
grep maxrepeat /etc/security/pwquality.conf
Fix
echo "maxrepeat = 3" >> /etc/security/pwquality.conf
Limit repeated characters in passwords.

Minimum Password Length

Check
grep minlen /etc/security/pwquality.conf
Fix
echo "minlen = 12" >> /etc/security/pwquality.conf
Enforce strong passwords.

SHA1 Disabled in Crypto Policies

Check
grep -i sha1 /etc/crypto-policies/state/current
Fix
update-crypto-policies --set FUTURE
Ensures SHA1 is disabled and only strong hashes are used.

Systemd Journal Remote Disabled

Check
systemctl is-enabled systemd-journal-remote
Fix
systemctl disable --now systemd-journal-remote
Remote journal logging should be disabled unless needed.

Systemd Journal Upload Disabled

Check
systemctl is-enabled systemd-journal-upload
Fix
systemctl disable --now systemd-journal-upload
Uploads to remote journald should be disabled unless required.

RSYSLOG FileCreateMode

Check
grep FileCreateMode /etc/rsyslog.conf
Fix
echo "FileCreateMode 0640" >> /etc/rsyslog.conf
Ensures new log files are created with correct permissions.

Login Banner /etc/issue

Check
cat /etc/issue
Fix
echo "Authorized users only!" > /etc/issue
Displays warning banner at login prompt.

Login Banner SSHD

Check
grep Banner /etc/ssh/sshd_config
Fix
sed -i 's|#Banner.*|Banner /etc/issue|' /etc/ssh/sshd_config
SSH login banner should match /etc/issue warning.

SSH ClientAliveCountMax

Check
grep ClientAliveCountMax /etc/ssh/sshd_config
Fix
sed -i 's/^#ClientAliveCountMax.*/ClientAliveCountMax 3/' /etc/ssh/sshd_config
Limits number of missed client alive messages before disconnect.

SSH ClientAliveInterval

Check
grep ClientAliveInterval /etc/ssh/sshd_config
Fix
sed -i 's/^#ClientAliveInterval.*/ClientAliveInterval 300/' /etc/ssh/sshd_config
Disconnect idle SSH sessions after interval.

Systemd Journald Compress

Check
grep Compress /etc/systemd/journald.conf
Fix
sed -i 's/^#Compress.*/Compress=yes/' /etc/systemd/journald.conf
Enables journal compression to save disk space.

Yama ptrace_scope

Check
cat /proc/sys/kernel/yama/ptrace_scope
Fix
echo 1 > /proc/sys/kernel/yama/ptrace_scope
Restrict ptrace to improve process security.

Disable IPv6 Router Advertisements

Check
sysctl net.ipv6.conf.all.accept_ra
Fix
sysctl -w net.ipv6.conf.all.accept_ra=0
Prevents accepting router advertisements in IPv6.

Disable Source Routing

Check
sysctl net.ipv4.conf.all.accept_source_route
Fix
sysctl -w net.ipv4.conf.all.accept_source_route=0
Disables source routing for security.

Disable ICMP Redirects

Check
sysctl net.ipv4.conf.all.accept_redirects
Fix
sysctl -w net.ipv4.conf.all.accept_redirects=0
Prevents man-in-the-middle attacks via ICMP redirects.

Enable SELinux Enforcing

Check
getenforce
Fix
setenforce 1
Enforces SELinux policies for security.

Enable Firewalld

Check
systemctl is-enabled firewalld
Fix
systemctl enable --now firewalld
Enables host firewall for protection.
