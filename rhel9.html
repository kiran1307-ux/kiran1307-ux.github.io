Secure SSH Parameters

PermitRootLogin no PasswordAuthentication no PubkeyAuthentication yes X11Forwarding no MaxAuthTries 3 ClientAliveInterval 300 ClientAliveCountMax 0

📊 Journald Logging

Storage=persistent Compress=yes ForwardToSyslog=yes

⚙️ Kernel Module Hardening

install usb-storage /bin/true install cramfs /bin/true install squashfs /bin/true

💾 Core Dump Restriction

* hard core 0 fs.suid_dumpable = 0

🔑 Password Quality

minlen = 14 dcredit = -1 ucredit = -1 lcredit = -1 ocredit = -1

🔐 Authentication (PAM)

auth required pam_faillock.so deny=5 unlock_time=900 account required pam_faillock.so

⏰ Cron Security

chmod 600 /etc/crontab chmod 700 /etc/cron.d

📁 File Permissions

chmod 600 /etc/passwd chmod 000 /etc/shadow

👤 File Ownership

chown root:root /etc/passwd chown root:root /etc/shadow

🌐 Network Kernel Parameters

net.ipv4.ip_forward = 0 net.ipv4.conf.all.accept_redirects = 0 net.ipv4.conf.all.secure_redirects = 0 net.ipv4.conf.all.log_martians = 1
