RHEL 9 Hardening Controls

SSH Hardening

File: /etc/ssh/sshd_config

PermitRootLogin no
Protocol 2
PermitEmptyPasswords no

Why: Protects SSH from unauthorized root access and weak authentication.

Journal Logs

journalctl --verify

Why: Ensures integrity of logs for auditing and troubleshooting.

File Ownership

chown root:root /etc/shadow

Why: Prevents unauthorized modifications of sensitive files.

File Permissions

chmod 600 /etc/shadow

Why: Limits access to critical files to authorized users only.

Authentication Hardening

authselect select sssd with-mkhomedir --force

Why: Centralizes authentication, ensures secure user management.

Storage Dump Prevention

echo 1 > /proc/sys/kernel/kptr_restrict

Why: Hides kernel pointers to unprivileged users.

Password Quality

/etc/security/pwquality.conf (minlen=12, dcredit=-1, ucredit=-1)

Why: Enforces strong passwords, reduces brute-force risks.

Kernel Hardening

sysctl -w net.ipv4.ip_forward=0
sysctl -w net.ipv4.conf.all.accept_redirects=0

Why: Enhances network security by disabling IP forwarding and ICMP redirects.

Modprobe Hardening

echo "install usb-storage /bin/true" > /etc/modprobe.d/disable-usb.conf

Why: Reduces attack surface by disabling unnecessary kernel modules.
