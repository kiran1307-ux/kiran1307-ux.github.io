About This Project

This Linux Hardening Guide provides practical security configurations for RHEL 8 and RHEL 9 systems.

It includes real-world controls based on industry standards such as:

  • SSH Security
  • Authentication & PAM
  • File Permissions
  • Kernel Hardening
  • Logging & Monitoring

This project is designed for learning, practice, and real-world server security implementation.
