📘 About This Project

This Linux Hardening Guide provides practical security configurations for RHEL 8 and RHEL 9 systems. It includes SSH, PAM, logging, kernel hardening, file permissions, and more. Designed for learning, practice, and real-world implementation.

✅ Purpose

Help users secure Linux servers following best practices.

🛠 Scope

Covers RHEL 8 & 9 configurations including SSH, PAM, cron, journald, kernel modules, and permissions.

🎯 Goal

Provide an interactive, professional learning platform for Linux security.
