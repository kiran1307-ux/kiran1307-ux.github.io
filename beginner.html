1. What is Linux?

Linux is an open-source operating system used in servers, cloud, and DevOps.

User → Shell → Kernel → Hardware
  • Kernel: Brain of OS
  • Shell: Command interface
  • Distros: Ubuntu, CentOS, RHEL

2. How to Install Linux

Options:
✔ VirtualBox (Recommended)
✔ Dual Boot
✔ Cloud VM (AWS)
  • Download Ubuntu ISO
  • Create VM
  • Start learning safely

3. File System

/
├── home
├── etc
├── var
├── bin
└── usr
  • /home: user files
  • /etc: configs
  • /var: logs
