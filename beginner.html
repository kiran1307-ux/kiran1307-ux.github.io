1. What is Linux?

Linux is an open-source operating system used in servers, cloud, and DevOps.

+----------------------+
|   Applications       |
+----------------------+
|       Shell          |
+----------------------+
|       Kernel         |
+----------------------+
|      Hardware        |
+----------------------+
  • Kernel: Core of OS
  • Shell: Interface to interact
  • Distros: Ubuntu, CentOS, RHEL

2. Linux Commands

Basic navigation commands:

pwd   → show current directory
ls    → list files
cd    → change directory

File operations:

cp file1 file2
mv file1 file2
rm file
mkdir folder

Permissions Diagram:

-rwxr-xr--
 | | | |
 | | | └── Others
 | | └──── Group
 | └────── Owner
 └──────── File type

3. Shell Scripting

Automate tasks using Bash scripts.

#!/bin/bash
echo "Hello World"

Flow:

Input → Script → Execution → Output
  • Variables
  • Loops (for, while)
  • Conditions (if-else)
