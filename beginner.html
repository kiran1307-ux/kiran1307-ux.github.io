Beginner Tutorials

Start your Linux & DevOps journey with structured tutorials covering commands, file systems, shell scripting, and foundational DevOps concepts.

1. Linux Basics

Learn what Linux is, its distributions, basic commands, and navigation of the file system.

  • Understanding Linux distributions (Ubuntu, CentOS, Fedora)
  • Terminal basics and shell commands
  • File system structure: /home, /etc, /var, /usr
  • File operations: ls, cd, cp, mv, rm, mkdir
  • Permissions: chmod, chown, chgrp

2. Shell Scripting

Automate tasks with Bash scripting.

  • Creating scripts: #!/bin/bash
  • Variables and user input
  • Conditionals: if, else, elif
  • Loops: for, while, until
  • Functions and script execution

3. Package Management

Learn to install, update, and remove software packages.

  • Debian-based systems: apt-get, apt
  • RedHat-based systems: yum, dnf
  • Checking installed packages and updates

4. Version Control with Git

Track code changes and collaborate using Git.

  • Installing Git and configuring user
  • Initializing repositories: git init
  • Cloning, adding, committing: git clone, git add, git commit
  • Branches: git branch, git checkout, git merge
  • Remote repositories: git remote, git push, git pull

5. Introduction to DevOps

Understand the fundamental DevOps concepts and tools for automation.

  • What is DevOps and why it matters
  • CI/CD pipeline basics
  • Introduction to Docker and containers
  • Monitoring and logging basics
