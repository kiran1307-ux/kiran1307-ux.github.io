1. What is Linux? 🖥️

Linux is an open-source operating system used to run servers, cloud infrastructure, and DevOps tools. Unlike Windows, Linux gives full control to users and is highly customizable.

It is widely used because it is stable, secure, and free. Most websites, applications, and cloud platforms run on Linux.

Architecture: User interacts with Shell → Shell communicates with Kernel → Kernel controls Hardware.

2. Installing Linux 💿

Installing Linux allows you to practice commands, run scripts, and manage servers safely.

Installation Options

  • Virtual Machine: Use VirtualBox or VMware inside Windows/macOS.
  • Dual Boot: Install Linux alongside Windows.
  • Cloud: Launch Linux servers online using AWS, Azure, or GCP.

Installation Steps

  1. Download Linux ISO (Ubuntu, CentOS, Fedora).
  2. Install VirtualBox and create VM.
  3. Attach ISO and boot VM.
  4. Follow installer prompts.
  5. Reboot and login to Linux.
3. Linux File System 🗂️

Linux uses a hierarchical file structure starting from /. Everything is treated as a file, including hardware devices.
