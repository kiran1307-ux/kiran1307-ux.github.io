RHEL 8 Hardening (CIS - 20 Controls)

Disable Root SSH Login

Check
grep PermitRootLogin /etc/ssh/sshd_config
Fix
echo "PermitRootLogin no" >> /etc/ssh/sshd_config systemctl restart sshd
Disable direct root SSH access.

Disable IP Forwarding

Check
cat /proc/sys/net/ipv4/ip_forward
Fix
echo "net.ipv4.ip_forward=0" >> /etc/sysctl.conf sysctl -w net.ipv4.ip_forward=0
Prevent system acting as router.

Disable cramfs

Check
lsmod | grep cramfs
Fix
echo "install cramfs /bin/true" > /etc/modprobe.d/cramfs.conf
Disable unused filesystem.

Password Minimum Length

Check
grep minlen /etc/security/pwquality.conf
Fix
echo "minlen=12" >> /etc/security/pwquality.conf
Enforce strong passwords.

Set UMASK

Check
umask
Fix
echo "umask 077" >> /etc/profile
Restrict default permissions.

GRUB Password

Check
grep password_pbkdf2 /boot/grub2/user.cfg
Fix
grub2-setpassword
Protect bootloader.

/etc/shadow Permissions

Check
ls -l /etc/shadow
Fix
chmod 600 /etc/shadow
Secure sensitive file.

/etc/shadow Ownership

Check
ls -l /etc/shadow
Fix
chown root:root /etc/shadow
Ensure root ownership.

Faillock Deny

Check
grep deny /etc/security/faillock.conf
Fix
echo "deny = 5" >> /etc/security/faillock.conf
Lock account after failures.

MaxAuthTries

Check
grep MaxAuthTries /etc/ssh/sshd_config
Fix
echo "MaxAuthTries 4" >> /etc/ssh/sshd_config
Limit login attempts.

LoginGraceTime

Check
grep LoginGraceTime /etc/ssh/sshd_config
Fix
echo "LoginGraceTime 60" >> /etc/ssh/sshd_config
Reduce brute-force window.

ClientAliveInterval

Check
grep ClientAliveInterval /etc/ssh/sshd_config
Fix
echo "ClientAliveInterval 300" >> /etc/ssh/sshd_config
Idle session timeout.

ClientAliveCountMax

Check
grep ClientAliveCountMax /etc/ssh/sshd_config
Fix
echo "ClientAliveCountMax 3" >> /etc/ssh/sshd_config
Limit retries.

Permit Empty Passwords

Check
grep PermitEmptyPasswords /etc/ssh/sshd_config
Fix
echo "PermitEmptyPasswords no" >> /etc/ssh/sshd_config
Disable empty passwords.

Disable X11 Forwarding

Check

Fix

Enable auditd

Check
systemctl status auditd
Fix
yum install audit -y systemctl enable --now auditd
Enable auditing.

Enable rsyslog

Check

Fix
systemctl enable --now rsyslog
Ensure logging.

ASLR (Randomize VA Space)

Check
sysctl kernel.randomize_va_space
Fix
sysctl -w kernel.randomize_va_space=2
Enable ASLR protection.
