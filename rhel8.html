Current UMASK for root

echo $UMASK

Why: Ensure root files have restrictive permissions.
Solution: Set UMASK in /etc/bashrc or /etc/profile: umask 077

Blacklist 'cramfs'

grep cramfs /etc/modprobe.d/*

Why: Prevent loading insecure/unneeded kernel modules.
Solution: Create a blacklist file: echo "install cramfs /bin/true" > /etc/modprobe.d/cramfs.conf

Files/Dirs in Home > 0750

find /home -type f -perm /077

Why: Detect overly permissive files.
Solution: Restrict permissions: chmod 750 /home/username/file

Fallock Deny Setting

grep deny /etc/security/faillock.conf

Why: Lock accounts after failed login attempts.
Solution: Add or edit: deny = 5 in /etc/security/faillock.conf

GRUB2 Password

grep -i "password_pbkdf2" /boot/grub2/user.cfg

Why: Prevent unauthorized boot modifications.
Solution: Set password: grub2-setpassword (will use pbkdf2 SHA512)

IP Forward

cat /proc/sys/net/ipv4/ip_forward

Why: Ensure system does not forward packets unless needed.
Solution: Disable: sysctl -w net.ipv4.ip_forward=0 and persist in /etc/sysctl.conf

pwquality MaxRepeat

grep maxrepeat /etc/security/pwquality.conf

Why: Limits repeated characters in passwords.
Solution: Set: maxrepeat = 3 in /etc/security/pwquality.conf

pwquality MinLen

grep minlen /etc/security/pwquality.conf

Why: Enforces minimum password length.
Solution: Set: minlen = 12 in /etc/security/pwquality.conf
