RHEL 8 Hardening Controls

SSH Hardening

File: /etc/ssh/sshd_config

PermitRootLogin no Protocol 2 PermitEmptyPasswords no

Why: Secures remote SSH access, disables root login, prevents empty passwords, enforces protocol 2.

Journal Logs

journalctl --verify

Why: Ensures log integrity for auditing and detecting tampering.

File Ownership

chown root:root /etc/shadow

Why: Prevents unauthorized modification of critical files.

File Permissions

chmod 600 /etc/shadow

Why: Restricts access to sensitive files to authorized users only.

Authentication Hardening

authconfig --passalgo=sha512 --update

Why: Uses SHA-512 for secure password hashing.

Storage Dump Prevention

echo 1 > /proc/sys/kernel/kptr_restrict

Why: Prevents exposure of kernel pointers to unprivileged users.

Password Quality

/etc/security/pwquality.conf (minlen=12, dcredit=-1, ucredit=-1)

Why: Enforces strong password complexity to prevent brute-force attacks.

Kernel Hardening

sysctl -w net.ipv4.ip_forward=0 sysctl -w net.ipv4.conf.all.accept_redirects=0

Why: Disables IP forwarding and ICMP redirects for network security.

Modprobe Hardening

echo "install usb-storage /bin/true" > /etc/modprobe.d/disable-usb.conf

Why: Disables unused kernel modules to reduce attack surface.
