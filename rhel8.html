RHEL 8 Hardening (CIS Controls)

Disable Root SSH Login

Check:

grep PermitRootLogin /etc/ssh/sshd_config

Fix:

echo "PermitRootLogin no" >> /etc/ssh/sshd_config systemctl restart sshd

Prevents direct root login via SSH.
CIS: Must be set to no.

Disable IP Forwarding

Check:

cat /proc/sys/net/ipv4/ip_forward

Fix:

echo "net.ipv4.ip_forward = 0" >> /etc/sysctl.conf sysctl -w net.ipv4.ip_forward=0

Stops system from acting as router.
CIS: Must be 0.

Disable cramfs Filesystem

Check:

lsmod | grep cramfs

Fix:

echo "install cramfs /bin/true" > /etc/modprobe.d/cramfs.conf

Disables unused filesystem.
Reduces attack surface.

Password Minimum Length

Check:

grep minlen /etc/security/pwquality.conf

Fix:

echo "minlen = 12" >> /etc/security/pwquality.conf

Enforces strong passwords.
CIS: Minimum 12 characters.

Set UMASK

Check:

umask

Fix:

echo "umask 077" >> /etc/profile

Restricts default file permissions.
CIS: Recommended 077.

GRUB2 Password

Check:

grep password_pbkdf2 /boot/grub2/user.cfg

Fix:

grub2-setpassword

Protects bootloader from tampering.
