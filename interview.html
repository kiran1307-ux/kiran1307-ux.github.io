Linux & DevOps Interview Questions (L2 & L3)

1. What will you do if a server is slow?
Check CPU, memory, disk, network.
top free -m iostat vmstat
2. Difference between soft link and hard link?
Soft link points to file path, hard link points to inode.
3. How to check which process is using a port?
ss -tulnp lsof -i :80
4. What happens when disk is full?
System may crash, logs stop, services fail.
5. How to find large files?
find / -type f -size +1G
6. What is load average?
Number of processes waiting for CPU.
7. Service not starting?
Check status, logs, config.
systemctl status journalctl -xe
8. What is Docker?
Containerization platform to run apps.
9. Pod CrashLoopBackOff?
kubectl logs kubectl describe pod
10. Website is down?
Check server → service → port → logs → network
11. High load but CPU idle?
Disk I/O issue
iostat -x 1
12. Docker container not starting?
docker logs docker inspect
13. CI/CD pipeline failed?
Check pipeline logs and failed stage.
14. Disk I/O high?
iostat iotop
15. Unauthorized access?
Check logs, block IP, secure SSH.
16. Pod not scheduling?
Resource issue or node selector.
17. Rolling deployment?
Gradual update without downtime.
18. Blue-green deployment?
Two environments switch traffic.
19. Memory leak debug?
top ps aux --sort=-%mem
20. Load balancer issue?
Check backend, logs, config.
