Advanced Linux & DevOps Interview (Product-Based Companies)

1. How do you handle a complete production outage?
Follow incident response:
Check alerts Check logs Check services Check infra (CPU, memory, disk) Rollback if needed
2. Explain how Kubernetes handles scaling?
Uses HPA (Horizontal Pod Autoscaler) based on CPU/memory metrics.
3. How do you debug memory leak in production?
top ps aux --sort=-%mem pmap PID
Check logs + heap usage.
4. Explain blue-green deployment in real scenario?
Two environments → switch traffic → rollback instantly if failure.
5. How do you secure a production server?
Disable root login, use SSH keys, firewall, patching, monitoring.
6. Kubernetes pod stuck in Pending?
kubectl describe pod
Check resources, node selector, taints.
7. How do you design a CI/CD pipeline?
Git → Build → Test → Docker → Deploy → Monitor
8. How do you monitor production systems?
Prometheus + Grafana + alerts.
9. Explain container orchestration?
Managing containers at scale using Kubernetes.
10. How do you troubleshoot high latency?
ping traceroute iostat
Check network, disk, backend.
11. Explain Infrastructure as Code with example?
Using tools like Terraform to create infra via code.
12. What happens inside docker run?
Pull image → create container → setup namespaces → run process.
13. How do you debug DNS issues?
dig nslookup cat /etc/resolv.conf
14. How do you handle zero downtime deployments?
Rolling updates, blue-green deployment.
15. Explain logging strategy in production?
Centralized logging using ELK stack.
